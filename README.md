# Minor Degree Allocation System

A Flask-based web application that automates the allocation of minor degree specializations based on student preferences and eligibility criteria.

---

## 🚀 Features

- Faculty Login Authentication
- Student Preference Handling
- Automated Allocation Logic
- Result Analysis & Display
- File Upload Support
- Clean Web Interface using HTML Templates

---

## 🛠️ Technologies Used

- Python
- Flask
- HTML/CSS
- Pandas
- Git & GitHub

## ▶️ How to Run the Project

1. Clone the repository:
```
git clone https://github.com/sayalisali123/minor-degree-allocation.git
```

2. Navigate into the project folder:
```
cd minor-degree-allocation
```

3. Install dependencies:
```
pip install -r requirements.txt
```

4. Run the Flask app:
```
python app.py
```

5. Open in browser:
```
http://127.0.0.1:5000
```
